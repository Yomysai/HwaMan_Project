 liste  des éléments demandés pour le projet "Hwa-Man" :

Fonctionnalités principales :

1. Recherche de mangas
2. Affichage des détails des mangas
3. Commentaires sur les mangas
4. Ajout de nouveaux mangas
5. Gestion des utilisateurs avec différents groupes et rôles
6. Gestion des groupes
7. Gestion des rôles
8. Gestion des autorisations
9. Pagination
10. Validation des données
11. Sécurité
12. Tests unitaires et tests d'intégration
13. Internationalisation (i18n)
14. Documentation

Architecture :

- Backend :
    
    - Framework : Spring Boot
    - Langage de programmation : Java
    - API RESTful : Pour la communication entre le frontend et le backend
    - Spring Data JPA : Pour l'accès à la base de données et la persistance des données
    - Base de données : MySQL
- Frontend :
    
    - Framework : React
    - Langage de programmation : JavaScript
    - Interface utilisateur : Interface réactive et attrayante

Modèle Conceptuel de Données (MCD) :

- Entités :
    - Users (Utilisateurs)
    - Roles (Rôles)
    - Teams (Groupes)
    - Items (Items)
    - Mangas (Mangas)
    - Books (Livres)
    - Comments (Commentaires)
    - UserRoles (Rôles des utilisateurs)
    - UserTeams (Groupes des utilisateurs)
    - GroupRoles (Rôles des groupes)
    - Pages (Pages)
    - Images (Images)
    - Categories (Catégories)

MCD Merise : Le MCD Merise représente les entités, leurs attributs et leurs relations dans la base de données. (AnalyseSI )  

Diagramme UML de structure : Le diagramme UML de structure représente la structure des classes et les relations entre elles. (ArgoUml)
